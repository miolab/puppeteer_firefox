# Puppeteer Firefox

GitHub (Puppeteer) https://github.com/puppeteer/puppeteer

---

# Install

```terminal
yarn add puppeteer
```

## Usage

```terminal
git clone https://github.com/miolab/puppeteer_firefox.git

cd puppeteer_firefox

node app.js
```

## version

- puppeteer / version "5.3.1"
- node / v12.16.1
